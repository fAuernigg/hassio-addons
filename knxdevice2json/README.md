# Home Assistant Add-on: KnxDevice 2 Json

Convert knx device of knxproj file to json and return result over mqtt.

![Supports aarch64 Architecture][aarch64-shield] ![Supports amd64 Architecture][amd64-shield] ![Supports armhf Architecture][armhf-shield] ![Supports armv7 Architecture][armv7-shield] ![Supports i386 Architecture][i386-shield]

## About

This Add-on allows to convert parameters and group addresses
of a specific knx device from knxproj to json format and return 
it over specified mqtt topic.
