# Home Assistant Add-on: Video Server based on ffmpeg

(Re)encode streams using ffmpeg and provide them on a specified port.

![Supports aarch64 Architecture][aarch64-shield] ![Supports amd64 Architecture][amd64-shield] ![Supports armhf Architecture][armhf-shield] ![Supports armv7 Architecture][armv7-shield] ![Supports i386 Architecture][i386-shield]

## About

This Add-on allows you to enable file sharing across different operating systems over
Reencode your video streams using ffmpeg and provide them to your a network.
